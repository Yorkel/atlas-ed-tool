How Many Topics Should We Look For?

Compare what the model finds when you change how many topics it looks for. The main model uses 30 topics. Here you can switch between 5, 15, and 30 topics - same data, same method - and see which topics survive and which disappear. Shared topics are shown in colour; topics that only exist in one version are shown in grey.

Comparison Result

What This Means

Core Finding Only 6% of topics remain when moving from 30 topics to 5. At 5 topics, one group absorbs almost half of all articles — school absence, mental health, free school meals, SEND, and exclusions all get lumped together. At 30, those become separate issues with different trends and different sources. The "zoom level" doesn't just add detail — it changes what counts as a finding. Specialist organisations like the Fischer Family Trust and Nuffield Foundation are invisible at low resolution. Their perspectives only appear when the model is set to find more topics. Lower resolution systematically silences minority voices.
Core Insight The number of topics acts like a zoom control — smaller voices only become visible as you zoom in. Only two topics survived every test: academy trust governance and Ofsted inspection reform. Everything else — the issues a policymaker might act on — changed depending on how many topics the model was asked to find.
