The Core Argument

Model specification choices — the number of topics (k), which sources are included, and how text is preprocessed — shape outputs as much as the data itself.

This dashboard demonstrates that there is no single "correct" topic model of education policy. Every model is a lens, and every lens has a focal point that makes some things visible while obscuring others.

What We Found

k is a volume knob for minority sources

At k=5, a single catch-all topic absorbs 49% of articles. Think tanks (FFT, EPI, Nuffield) are invisible. At k=30, they each get dedicated topics. The choice of k determines which voices are heard.

The education debate is produced by media

Removing SchoolsWeek (70% of the corpus) doesn't just reduce the dataset — it fundamentally changes what education policy "looks like". Teacher strikes vanish. Shared debate fragments into institutional monologues. Crisis narratives become procedural compliance. The full corpus is crisis-oriented (council deficits, EHCP wait times); the no-media corpus is procedural (statutory guidance, compliance requirements).

Contestability reveals uncertainty

The contestability score (normalised Shannon entropy) shows that many articles don't belong clearly to a single topic. High contestability articles sit at the boundaries between topics — precisely the articles where specification choices matter most.

SpecCheck: A Vision for Specification Transparency

AtlasED demonstrates a methodology we call SpecCheck — the practice of systematically varying model specifications and presenting the differences to users.

Rather than presenting a single set of results as definitive, SpecCheck asks: what would change if we made different choices? This approach treats model specification as an analytical decision that should be visible, not hidden.

Future applications could extend this to other NLP tasks: sentiment analysis, named entity recognition, document classification — any task where preprocessing and parameter choices shape outputs.

Data & Methods

  • Period: January 2023 – September 2025
  • Sources: SchoolsWeek (media), Gov.uk (government), EPI, Nuffield, FFT (think tanks), FED (federation)
  • Method: Non-negative Matrix Factorisation (NMF) with TF-IDF
  • Parameters: init=nndsvd, random_state=42, max_iter=1000
  • TF-IDF: min_df=3, max_df=0.85, max_features=3000, ngram_range=(1,2)
  • Election split: 2024-07-04 (UK General Election)
