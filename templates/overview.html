About This Tool

What Choices Shape What AI Finds?

AI is shaping real decisions — but the choices behind these systems aren't visible. Builders can't easily show them, decision makers can't assess them, and those affected can't question them. AtlasED analyses thousands of policy documents to show how design choices shape what AI finds.

Why This Matters A policymaker asks the biggest issue in education. The model says "child welfare," so funding follows. Change one model setting, and that single topic splits into distinct problems — each needing different solutions. The answer wasn't wrong — it reflected one way the model was set up.

How It Works

1
Data collection
5,000+ documents across six organisation types. What's included shapes what's found.
2
Text processing
Text is cleaned and simplified. These choices shape the model's language.
3
Model
We use NMF — a transparent method with visible settings. In more complex AI, these choices still exist but are harder to see.
4
Quality checks
The model is consistent and passes standard tests. But these don't show how results change under different choices.
The Core Finding Fewer than 1 in 5 findings stayed the same across configurations — while every quality check still passed. The model looked just as reliable each time. It was just telling a different story.
