About This Tool

What Choices Shape What AI Finds?

AI is shaping real decisions — but the choices behind these systems aren't visible. Builders can't easily show them, decision makers can't assess them, and those affected can't question them.

AtlasED makes those choices visible. It analyses thousands of policy documents to show how design decisions shape what AI finds.

Why This Matters A policymaker asks AI what the biggest issue in education is. The model says 'learning outcomes,' so funding follows. Change one setting, and that single category splits into reading scores, maths attainment, progress rates, and early years development — each needing different action. The result wasn't wrong. It depended on how the question was set up.

The Main Model

The model used here is a straightforward machine learning method that reads documents, groups them, and identifies topics. We chose it deliberately — because every choice is visible. The number of topics, the data sources, the text processing — change any one and watch the findings shift. Here you can see what the model found — and what it would find if different choices were made.
