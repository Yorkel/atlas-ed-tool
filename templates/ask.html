What Happens When the Data Changes?

The previous page changed the model. This page changes the data. Education media (SchoolsWeek) makes up 70% of the corpus. What happens when we remove it entirely and retrain the same model?

How to read this Both models use k=30 and the same method. The only difference is the data: one includes all sources, the other removes media. Topics that appear in both share the same colour. Grey topics only exist in one version — they depend on which voices are included.

What Changed
Key Finding Removing SchoolsWeek doesn't just shrink the dataset — it changes what education policy "looks like." Teacher strikes vanish. Shared debate fragments into institutional monologues. The full corpus is crisis-oriented; the no-media corpus is procedural.
